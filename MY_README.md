### 一键部署
1. 下载代码
```
git clone https://github.com/WeisonWEileen/unitree_ros.git
```

2. 打开vsode
```
code unitree_ros
```

3. 在vscode中安装 `dev container`插件
4. 按下 `ctrl + shift + p`,然后选择
`reopen  in container`选项。
